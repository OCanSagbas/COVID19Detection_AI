# 🦠 COVID-19 Detection using AI
Bu proje, Yapay Zeka (AI) tekniklerini kullanarak COVID-19 hastalığını tespit etmeye yönelik bir uygulamadır.

# 📄 Proje Açıklaması
COVID-19 tespiti için makine öğrenimi (ML) ve derin öğrenme (DL) teknikleri kullanılmıştır. Projede, çeşitli veri setlerinden faydalanarak sınıflandırma modelleri eğitilmiş ve test edilmiştir.

# 🚀 Özellikler
Veri Analizi: COVID-19 verilerini analiz etme ve ön işleme.
Model Eğitimi: Derin öğrenme modelleri ile COVID-19 tespiti.
Sonuç Değerlendirme: Model performansının doğruluk, hassasiyet ve F1 skor metrikleriyle ölçümü.
